

# LXP (Learning Experience Platform) 프로젝트

> 체계적인 학습 경험을 제공하는 상업용 온라인 강의 플랫폼

안녕하세요! LXP 프로젝트 팀에 오신 것을 환영합니다.

본 문서는 LXP 프로젝트의 목적, 주요 기능, 그리고 우리가 지향하는 개발 철학을 공유하기 위한 팀 가이드입니다.

저희는 사용자가 쉽게 배우고, 강사가 편하게 콘텐츠를 관리하며, 관리자가 효율적으로 플랫폼을 운영할 수 있는 시스템을 구축하는 것을 목표로 합니다.

---

## 🧭 프로젝트 개요

저희는 Java와 Spring Boot를 기반으로 하는 **상업용 온라인 강의 플랫폼(LXP)**을 개발합니다.

사용자는 강의를 구매하여 수강하고, 강사는 자신의 강의를 등록하여 판매할 수 있습니다. 관리자는 이 모든 과정을 관리하는 백오피스 시스템을 가집니다.

* **View 방식:** 별도의 프론트엔드(React, Vue 등) 없이, `Thymeleaf`를 활용한 **서버 사이드 렌더링(SSR)** 방식을 채택하여 백엔드에서 화면 렌더링까지 담당합니다.

---

## 🚀 주요 기능

우리가 구현할 핵심 기능은 다음과 같습니다.

### 👩‍🎓 사용자 (학생) 기능
* **회원 관리:** 회원가입 및 로그인
* **강좌 탐색:** 전체 강좌 목록 조회 (카테고리별, 조회순 등)
* **결제:** 장바구니 기능, 강좌 구매
* **학습:** 나의 강의실 (수강중인 강좌 목록), 강좌별 진도율(%) 확인

### 🧑‍🏫 강사 (Instructor) 기능
* **콘텐츠 관리:** '나의 강좌 관리' 페이지
* **강좌 개설:** 신규 강좌 등록, 설명, 가격 등 정보 수정
* **강의 관리:** 강좌에 포함될 개별 강의(비디오) 업로드 및 순서 편집

### 🤝 커뮤니티 기능
* **소통:** 자유 소통 게시판 (후기, 질문, 스터디 등)
* **상호작용:** 게시글 작성/수정/삭제, 댓글 및 대댓글 기능

### 🔐 관리자 (Admin) 기능
* **백오피스:** 관리자 전용 대시보드
* **사용자 관리:** 전체 회원 조회 및 권한(역할) 관리
* **콘텐츠 관리:** 모든 강좌 및 강의 제어
* **운영 관리:** 결제 내역, 통계 등 확인

---


## 💻 주요 기술 스택

* **Backend:** Java, Spring Boot
* **Database:** MySQL
* **View:** Thymeleaf
* **Data Access:** Spring Data JPA
* **Security:** Spring Security
